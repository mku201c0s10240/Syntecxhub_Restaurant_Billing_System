A simple Command-Line Interface (CLI) based Restaurant Billing System built using Core Java and Object-Oriented Programming (OOP) concepts.
This project allows users to manage restaurant menu items, place orders with quantities, and generate itemized bills with GST calculation.

# Features
1. Add new menu items dynamically (ID, Name, Price)
2. Remove menu items by ID
3. Display available menu items
4. Place orders with quantity selection
5. Remove items from the order
6. Generate itemized bills
7. GST (5%) calculation
8. User-friendly CLI menu
9. Beginner-friendly and modular code structure
